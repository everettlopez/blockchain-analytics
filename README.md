# Blockchain Analytic Dashboard
This application develops a personalized financial portfolio dashboard allowing individuals to track their personal digital asset holdings, as well as a get real-time access to token prices and transaction history on certain blockchains.

## Data Sources
- [Alchemy API](https://www.alchemy.com/)
- [Coin Market Cap API](https://coinmarketcap.com/api/)
